Запуск проекта

1. Клонирование репозитория
Сначала клонируйте репозиторий с проектом:
git clone <URL вашего репозитория>
cd <название папки вашего репозитория>

2.Установите все необходимые зависимости.
Для этого можно использовать pip:
pip install -r requirements.txt

3. Запуск сервера
Запустите сервер разработки FastAPI с помощью uvicorn:
uvicorn main:app --reload

Сервер будет доступен по адресу http://127.0.0.1:8000

Использование API эндпоинтов

Эндпоинты для учеников

1. Список всех учеников

URL: /students/
Метод: GET

2. Создать нового ученика

URL: /students/
Метод: POST
Тело запроса:
{
    "name": "John",
    "surname": "Doe"
}

3. Получить информацию о конкретном ученике

URL: /students/{student_id}
Метод: GET

4. Обновить информацию о конкретном ученике

URL: /students/{student_id}
Метод: PATCH
Тело запроса:
{
    "name": "Jane",
    "surname": "Doe"
}

5. Удалить ученика

URL: /students/{student_id}
Метод: DELETE

Эндпоинты для оценок

1. Добавить оценку ученику

URL: /score/
Метод: POST
Тело запроса:
{
    "student_id": 1,
    "score": 90
}

2. Получить информацию о конкретной оценке

URL: /score/{score_id}
Метод: GET

3. Обновить информацию о конкретной оценке

URL: /score/{score_id}
Метод: PATCH

4. Удалить оценку

URL: /score/{score_id}
Метод: DELETE

Взаимодействие с API можно с помощью Postman
